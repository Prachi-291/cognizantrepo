# Exercise 4: Employee Management System

## Aim
To implement an Employee Management System using arrays.

## Problem Statement
Create a Java program to manage employee records using an array.

## Operations Performed
- Add Employee
- Search Employee
- Display Employees
- Delete Employee

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Add | O(1) |
| Search | O(n) |
| Traverse | O(n) |
| Delete | O(n) |

## Advantages of Arrays
- Simple implementation
- Fast access using index
- Suitable for fixed-size collections

## Limitations
- Fixed size
- Insertion and deletion require shifting elements

## Conclusion
Arrays are suitable when the number of records is known in advance and random access is required.
