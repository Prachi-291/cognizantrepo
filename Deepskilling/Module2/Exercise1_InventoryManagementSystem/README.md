# Exercise 1: Inventory Management System

## Aim

To develop an Inventory Management System using Java and HashMap for efficient storage and retrieval of product information.

## Problem Statement

Design an inventory system that supports adding, updating, searching, deleting, and displaying products.

## Data Structure Used

- HashMap<Integer, Product>

A HashMap provides fast access to products using Product ID as the key.

## Operations

- Add Product
- Search Product
- Update Product
- Delete Product
- Display Inventory

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Add | O(1) |
| Search | O(1) |
| Update | O(1) |
| Delete | O(1) |

## Advantages

- Fast lookup using Product ID
- Efficient insertion and deletion
- Suitable for large inventories

## Conclusion

Using HashMap improves the overall efficiency of inventory management by providing constant-time access to product records.
