# Module 9 - React

This module contains hands-on React exercises covering components,
lifecycle, ES6, event handling, conditional rendering, list rendering
and API integration.

## Q1 - First React Application

**Project:** `myfirstreact1`\
**Objective:** Create and run the first React application.\
**Concepts:** React, Components, Virtual DOM, Create React App.

## Q2 - Student Management Portal

**Project:** `studentapp2`\
**Objective:** Create multiple class components (Home, About, Contact).\
**Concepts:** Class Components, render(), Component Rendering.

## Q3 - Student Score Calculator

**Project:** `scorecalculatorapp`\
**Objective:** Display student details using props and calculate
percentage.\
**Concepts:** Functional Components, Props, CSS.

## Q4 - Blog Application

**Project:** `blogapp`\
**Objective:** Fetch and display posts from an API.\
**Concepts:** State, Lifecycle Methods, Fetch API.

## Q9 - ES6 Cricket App

**Project:** `cricketapp`\
**Objective:** Display player details using ES6 features.\
**Concepts:** Arrow Functions, Destructuring, map(), Components.

## Q10 - Office Space Rental App

**Project:** `officespacerentalapp`\
**Objective:** Display office space details with images and rent-based
conditional styling.\
**Concepts:** JSX, Props, Conditional Styling, Components.

## Q11 - Event Examples App

**Project:** `eventexamplesapp`\
**Objective:** Implement React event handling and currency conversion.\
**Concepts:** onClick, onChange, onSubmit, Synthetic Events, State.

## Q12 - Ticket Booking App

**Project:** `ticketbookingapp`\
**Objective:** Implement login/logout using conditional rendering.\
**Concepts:** Conditional Rendering, useState, Props, Element Variables.

## Q13 - Blogger App

**Project:** `bloggerapp`\
**Objective:** Display Course, Book and Blog details.\
**Concepts:** Components, List Rendering, map(), Keys, Conditional
Rendering.
