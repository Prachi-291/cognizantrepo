# Exercise 6: Library Management System

## Aim
To implement Linear Search and Binary Search algorithms for searching books in a library.

## Problem Statement
Develop a Java program that allows users to search for books by title using Linear Search and Binary Search.

## Algorithms Used

### Linear Search
- Searches each element one by one.
- Suitable for small or unsorted datasets.

### Binary Search
- Searches by repeatedly dividing the sorted array into halves.
- Requires the array to be sorted before searching.

## Time Complexity

| Algorithm | Best Case | Average Case | Worst Case |
|-----------|-----------|--------------|------------|
| Linear Search | O(1) | O(n) | O(n) |
| Binary Search | O(1) | O(log n) | O(log n) |

## Conclusion
Binary Search is more efficient for large sorted datasets, while Linear Search is useful for small or unsorted collections.
