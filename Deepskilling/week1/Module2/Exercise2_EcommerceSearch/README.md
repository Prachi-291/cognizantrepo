# Exercise 2: E-commerce Platform Search Function

## Aim
To implement Linear Search and Binary Search algorithms for searching products in an e-commerce platform.

## Problem Statement
Develop a search system to find products using their product name.

## Algorithms Used

### Linear Search
- Checks each element one by one.
- Suitable for small or unsorted datasets.

### Binary Search
- Searches by repeatedly dividing the search space into halves.
- Requires the array to be sorted.

## Time Complexity

| Algorithm | Best | Average | Worst |
|-----------|------|---------|-------|
| Linear Search | O(1) | O(n) | O(n) |
| Binary Search | O(1) | O(log n) | O(log n) |

## Conclusion
Binary Search is more efficient than Linear Search for large sorted datasets, while Linear Search is suitable for smaller or unsorted collections.
