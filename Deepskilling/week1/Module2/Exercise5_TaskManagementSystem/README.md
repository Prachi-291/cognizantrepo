# Exercise 5: Task Management System

## Aim

To implement a Task Management System using a Singly Linked List.

## Problem Statement

Create a Java program to manage tasks using a Singly Linked List.

## Operations Performed

- Add Task
- Search Task
- Display Tasks
- Delete Task

## Linked List

A Singly Linked List consists of nodes where each node stores data and a reference to the next node.

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Add | O(n) |
| Search | O(n) |
| Traverse | O(n) |
| Delete | O(n) |

## Advantages

- Dynamic memory allocation
- Easy insertion and deletion
- No fixed size limitation

## Conclusion

Linked Lists are suitable when the number of elements changes frequently.
